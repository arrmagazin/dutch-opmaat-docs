# Specifiers  

## Express equality/sameness

> Use `als` (not `dan`!) to say two things are equal.

| Pattern | Meaning | Example |
|---------|---------|---------|
| `zo` | so | Deze tas is **zo** zwaar. (This bag is so heavy.) |
| `zomaar` | evenly |  |
| `zoals` | such as / like | Talen **zoals** Nederlands en Duits. |
| `als` / `zulk` peer | as | Hij werkt *als* docent. |
| `even … als` | as … as | *Hij is `even` groot `als` ik.* |
| `net zo … als` | just as … as | *Zij loopt `net zo` snel `als` hij.* |

## Degree

Degree adverbs go **before** the adjective or adverb they scale.

| Dutch | English | Example |
|-------|---------|---------|
| `helemaal` | completely | Ik ben het **helemaal** vergeten. |
| `te` / `veel te` | too (excessive) / over too | Dat is **te** duur. (That's too expensive.) |
| `bijna` | almost | Ik ben **bijna** klaar. (I'm almost done.) |
| `meest` | most |  |
| `vooral` | mainly, before all |  |
| `erg` | very | Het is `erg` mooi. (It's very beautiful.) |
| `heel` | strongly / very | Het is `heel` mooi. (It's very beautiful.) |
| `zeer` | very (formal) | Wij zijn **zeer** tevreden. (We are very satisfied.) |
| `best` | quite / pretty | De film was **best** goed. (The film was quite good.) |
| `nogal` | rather | Het is **nogal** druk in de stad. (It's rather busy in town.) |
| `behoorlijk` | considerably | Het is **behoorlijk** koud vandaag. (It's considerably cold today.) |
| `redelijk` | fairly | Het was **redelijk** druk. (It was fairly busy.) |

> For the *softening* degree words — `nogal`, `best`, `erg`, `een beetje` — see the toners page; they downplay where `heel` boosts.

## Amount

| Dutch | English | Example |
|-------|---------|---------|
| `een beetje` | a little | Ik ben **een beetje** moe.  |
| `genoeg` | enough | We hebben **genoeg** brood voor iedereen.  |

> niets -> bijna niets -> wenig -> genoeg -> veel -> heel veel -> te veel -> veel te veel.

## Frequency

| Dutch | English | Example |
|-------|---------|---------|
| `altijd` | always | Ik drink **altijd** koffie in de ochtend. (I always drink coffee in the morning.) |
| `telkens` | every time | Hij komt **telkens** te laat. (He is late every time.) |
| `elke` (`dag` / `keer`) | every (day / time) | Ik drink **elke dag** thee. (I drink tea every day.) |
| `alledaags` | everyday | Dat is een **alledaags** probleem. (That's an everyday problem.) |
| `meestal` | usually | Ik werk **meestal** thuis. (I usually work from home.) |
| `regelmatig` | regularly | Ik leer **regelmatig** taal. |
| `vaak` / `dikwijls` | often | Ik ga **vaak** naar de markt. (I often go to the market.) |
| `soms` | sometimes | **Soms** regent het. (Sometimes it rains.) |
| `af en toe` | now and then | Ik drink **af en toe** een glas wijn. (I drink a glass of wine now and then.) |
| `zelden` | rarely | Hij komt **zelden** op tijd. (He's rarely on time.) |
| `alweer` | again | Ben je **alweer** te laat? (Are you late again?) |
| `nogmaals` / `nog eens` | one more time | **Nogmaals** dank en groet. |
| `een keer` | once | Ik ben er **een keer** geweest. (I've been there once.) |
| `weleens` | at least once |  |
| `bijna nooit` | almost never | Hij belt **bijna nooit** op. (He almost never calls.) |
| `nooit` | never | Ik ben **nooit** in Parijs geweest. (I've never been to Paris.) |

## Approximation

| Dutch | English | Example |
|-------|---------|---------|
| `precies` | exactly | Dat is **precies** wat ik bedoel. (That's exactly what I mean.) |
| `ongeveer` | approximately | Het duurt **ongeveer** een uur. (It takes approximately an hour.) |
| `circa/ca.` | approximately (formal, written) | De reis kost **circa** 200 euro. (The trip costs approximately 200 euros.) |
| `rond` | around (time, numbers) | Ik kom **rond** acht uur. (I'll come around eight o'clock.) |
| `zo` | about, roughly (informal) | Er waren **zo'n** twintig mensen. (There were about twenty people.) |
| `een stuk of` | about (countable, informal) | Geef me **een stuk of** vijf. (Give me about five.) |
| `ruim` | a little over, more than | Het duurde **ruim** een uur. (It took a little over an hour.) |
| `krap` | just under, barely | We hebben **krap** een week. (We have just under a week.) |
