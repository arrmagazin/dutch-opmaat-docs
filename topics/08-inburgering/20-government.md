# government and administration

Words and phrases for talking about government, elections and voting.

## Woordenschat — Vocabulary

- [ ] Ik praat niet graag over **politiek**.
- [ ] De **regering** neemt een nieuw besluit.
- [ ] De **minister** komt op televisie.
- [ ] De **premier** houdt een toespraak.
- [ ] Welke **partij** vind jij goed?
- [ ] De **verkiezingen** zijn volgende week.
- [ ] Morgen ga ik **stemmen**.
- [ ] Mijn **stem** is belangrijk.
- [ ] De wet komt in het **parlement**.
- [ ] De nieuwe **wet** geldt vanaf januari.
- [ ] Elke **burger** mag stemmen.
- [ ] Ik werk voor de **gemeente**.
- [ ] De **burgemeester** opent het feest.
- [ ] We betalen elk jaar **belasting**.
- [ ] Ons **land** is klein maar mooi.
- [ ] De **koning** bezoekt de stad.
- [ ] Nederland is een **democratisch** land.
- [ ] Vanavond is er een **debat** op tv.

## Uitdrukkingen — Common phrases

- [ ] **Ik ga stemmen** op zondag.
- [ ] **Op wie stem jij** dit jaar?
- [ ] **Ik ben het ermee eens**.
- [ ] **Ik ben het er niet mee eens**.
- [ ] **Dat is een goed plan** voor de stad.
- [ ] **Wat vind jij ervan**?
- [ ] We moeten dit jaar veel **belasting betalen**.
- [ ] Ik moet **naar de gemeente gaan** voor een paspoort.
- [ ] Het parlement wil een nieuwe **wet aannemen**.
- [ ] Welke partij is nu **aan de macht**?

Words and phrases for filling in forms and planning appointments in your diary.

- [ ] Ik wil graag **een afspraak maken**.
- [ ] Ik ga de datum even **noteren**.
- [ ] Wilt u **hieronder tekenen**?
- [ ] **Wat is de datum** van de afspraak?
- [ ] Ik zet het **in mijn agenda**.
- [ ] **Vul hier uw gegevens in**, alstublieft.
- [ ] **Past het u volgende week** dinsdag?
- [ ] Ik moet **de afspraak verzetten**.
- [ ] Ik moet **even in de kalender kijken**.
- [ ] Wanneer kunt u komen en wanneer past het voor u?
- [ ] Wat moet ik doen? Wat moet ik meenemen?
- [ ] Zullen we op 5 juli om 10.00 uur of om 15.00 uur afspreken?

## Administratie & overheid — Admin & government

- [ ] Een paspoort is **verplicht** voor de reis.
- [ ] Ik wil een uitkering **aanvragen**.
- [ ] Je moet je **inschrijven** bij de gemeente.
- [ ] Kunt u uw nieuwe adres **doorgeven**?
- [ ] Ik kan niet **inloggen** op mijn account.
- [ ] Je hebt **toestemming** van je ouders nodig.
- [ ] Hij krijgt elke maand een **uitkering**.
- [ ] Ik moet eerst mijn **eigen risico** betalen.
- [ ] Mag ik uw **rijbewijs** zien?
- [ ] Mijn **paspoort** is verlopen.
- [ ] Neem een geldig **identiteitsbewijs** mee.
- [ ] We gaan in de zomer **trouwen**.
- [ ] Mijn ouders gaan **scheiden**.
- [ ] Mijn dochter gaat naar de **kinderopvang**.
- [ ] We ontvangen elk kwartaal **kinderbijslag**.
- [ ] Studenten kunnen **studiefinanciering** aanvragen.
