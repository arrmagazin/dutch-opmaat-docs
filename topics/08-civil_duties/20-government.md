# Politics

Words and phrases for talking about government, elections and voting.

## Woordenschat — Vocabulary

- [ ] Ik praat niet graag over **politiek**.
- [ ] De **regering** neemt een nieuw besluit.
- [ ] De **minister** komt op televisie.
- [ ] De **premier** houdt een toespraak.
- [ ] Welke **partij** vind jij goed?
- [ ] De **verkiezingen** zijn volgende week.
- [ ] Morgen ga ik **stemmen**.
- [ ] Mijn **stem** is belangrijk.
- [ ] De wet komt in het **parlement**.
- [ ] De nieuwe **wet** geldt vanaf januari.
- [ ] Elke **burger** mag stemmen.
- [ ] Ik werk voor de **gemeente**.
- [ ] De **burgemeester** opent het feest.
- [ ] We betalen elk jaar **belasting**.
- [ ] Ons **land** is klein maar mooi.
- [ ] De **koning** bezoekt de stad.
- [ ] Nederland is een **democratisch** land.
- [ ] Vanavond is er een **debat** op tv.

## Uitdrukkingen — Common phrases

- [ ] **Ik ga stemmen** op zondag.
- [ ] **Op wie stem jij** dit jaar?
- [ ] **Ik ben het ermee eens**.
- [ ] **Ik ben het er niet mee eens**.
- [ ] **Dat is een goed plan** voor de stad.
- [ ] **Wat vind jij ervan**?
- [ ] We moeten dit jaar veel **belasting betalen**.
- [ ] Ik moet **naar de gemeente gaan** voor een paspoort.
- [ ] Het parlement wil een nieuwe **wet aannemen**.
- [ ] Welke partij is nu **aan de macht**?
