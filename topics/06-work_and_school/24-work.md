# Profession and Work

Words and phrases for talking about jobs, colleagues, and the workplace.

## Woordenschat — Vocabulary

- [ ] Wat is jouw **beroep**?
- [ ] Ik hou van mijn **werk**.
- [ ] Zij heeft een nieuwe **baan**.
- [ ] Ik **werk** bij een groot bedrijf.
- [ ] Mijn **collega** is heel aardig.
- [ ] De **baas** is vandaag niet op kantoor.
- [ ] Ik ga met de fiets naar **kantoor**.
- [ ] Het **salaris** is elke maand op tijd.
- [ ] Ik wil **solliciteren** op deze baan.
- [ ] Mijn **sollicitatie** was succesvol.
- [ ] Ik teken morgen mijn **contract**.
- [ ] De **vergadering** duurt een uur.
- [ ] Hij heeft veel **ervaring** met computers.
- [ ] Elke **werknemer** krijgt een laptop.
- [ ] Mijn **werkgever** betaalt de cursus.
- [ ] Ik zoek een **functie** in de zorg.
- [ ] De **dokter** werkt in het ziekenhuis.
- [ ] Onze **leraar** geeft Nederlandse les.

## Uitdrukkingen — Common phrases

- [ ] **Wat voor werk doe je** eigenlijk?
- [ ] **Ik werk als** verpleegkundige.
- [ ] **Ik werk parttime**, drie dagen per week.
- [ ] **Ik werk fulltime** bij een bank.
- [ ] Eindelijk **heb ik een baan gevonden**!
- [ ] **Hoe laat begint je werk** morgen?
- [ ] **Ik werk van negen tot vijf** op kantoor.
- [ ] Let op, **de deadline is vrijdag**.
- [ ] Om tien uur **heb ik een vergadering**.
- [ ] **Ik stuur mijn cv op** naar het bedrijf.
- [ ] Ik wil op 5 juli een dag vrij hebben, omdat ik ziek ben.
- [ ] Ik wil op zondag vrij omdat ik familie bezoek, en volgende zaterdag wil ik vrij zijn om uit te slapen.
- [ ] Ik mail omdat ik mijn dienst wil ruilen, omdat ik ziek ben.
- [ ] Ik moet vandaag werken.
- [ ] Ik neem elke dag twee keer pauze. Tijdens mijn pauze drink ik koffie.
- [ ] Ik werk al twee jaar fulltime als dokter in Den Haag, veertig uur per week.
- [ ] Ik werk in een winkel. Ik maak de vloeren schoon. Ik vind dat fijn werk.
- [ ] Ja, ik heb veel vergaderingen. Ik vind dat fijn.
- [ ] Hij maakt zich klaar. Hij kijkt naar zijn werk.
- [ ] Hij werkt met haar. Hij maakt de vloer schoon.
- [ ] Zij maakt zich klaar. Zij sluit de deur. Daarna werkt zij in het park.

## Solliciteren & beroepen — Applying & professions

- [ ] **De ingenieur** ontwerpt de brug.
- [ ] **De ondernemer** start een eigen bedrijf.
- [ ] **De verkoper** in de winkel hielp me goed.
- [ ] **De werktijden** zijn van negen tot vijf.
- [ ] Volgende week heb ik **verlof**.
- [ ] Ik heb gereageerd op **de vacature**.
- [ ] Ik schrijf een goede **sollicitatiebrief**.
- [ ] Ik hoop **aangenomen te worden** voor de baan.
