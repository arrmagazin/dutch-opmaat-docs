# Education

Words and phrases for talking about school, studying, and education in the Netherlands.

## Woordenschat — Vocabulary

- [ ] Het **onderwijs** in Nederland is goed.
- [ ] Mijn dochter zit op de **basisschool**.
- [ ] Hij gaat na de zomer naar de **middelbare school**.
- [ ] Zij studeert aan de **universiteit** van Amsterdam.
- [ ] De **student** woont op kamers in de stad.
- [ ] Onze **docent** legt alles rustig uit.
- [ ] Ik volg een **opleiding** tot verpleegkundige.
- [ ] Ik doe een **cursus** Nederlands.
- [ ] Na drie jaar krijg je je **diploma**.
- [ ] Morgen heb ik een **examen** wiskunde.
- [ ] Ik heb een goed **cijfer** voor de toets.
- [ ] Ik moet vanavond hard **studeren**.
- [ ] Kinderen **leren** snel een nieuwe taal.
- [ ] Aardrijkskunde is mijn favoriete **vak**.
- [ ] Het **college** begint om negen uur.
- [ ] In de **bibliotheek** is het altijd stil.
- [ ] Je moet je op tijd **inschrijven** voor de cursus.
- [ ] Het **collegegeld** is dit jaar hoger.
- [ ] Onze **school** begint om half negen.
- [ ] Ik zit in **klas** drie.
- [ ] De **leraar** legt de som uit.
- [ ] De **lerares** geeft Engels.
- [ ] Elke **leerling** heeft een laptop.
- [ ] De **les** duurt vijftig minuten.
- [ ] Ik maak mijn **huiswerk** na het eten.
- [ ] Sla je **boek** open op bladzijde tien.
- [ ] Ik schrijf de woorden in mijn **schrift**.
- [ ] Mag ik jouw **pen** even lenen?
- [ ] Teken de lijn met een **potlood**.
- [ ] De juf schrijft op het **bord**.
- [ ] Ik heb een goed **cijfer** voor wiskunde.
- [ ] Morgen hebben we een **toets** aardrijkskunde.
- [ ] In de **pauze** eet ik mijn boterham.
- [ ] Mijn **rooster** hangt aan de muur.
- [ ] Mijn favoriete **vak** is geschiedenis.
- [ ] De kinderen spelen op het **schoolplein**.
- [ ] Mijn boeken zitten in mijn **rugzak**.

## Uitdrukkingen — Common phrases

- [ ] **Wat studeer je** aan de universiteit?
- [ ] **Ik zit in het eerste jaar** van mijn opleiding.
- [ ] Goed nieuws: **ik ben geslaagd** voor mijn examen!
- [ ] Helaas **ben ik gezakt** voor wiskunde.
- [ ] Ik moet vanavond nog **huiswerk maken**.
- [ ] Tijdens het college **maak ik aantekeningen**.
- [ ] Ik hoop een hoog **cijfer te halen**.
- [ ] Ik ga vanavond **voor een toets leren**.
- [ ] **Wanneer is de toets** van Nederlands?
- [ ] Sorry dat **ik te laat ben**.
- [ ] **Zullen we samen werken** aan de opdracht?
- [ ] **Mag ik naar de wc**, alsjeblieft?
- [ ] **Vergeet je huiswerk niet** voor morgen.
- [ ] **Goed gedaan**, je hebt alles goed!
- [ ] Hoeveel kost het per maand en hoeveel kosten de lessen?
- [ ] Wanneer begint de cursus?
- [ ] Wanneer is er plaats?
- [ ] Wanneer kan ik de toets maken?
- [ ] Wat moet zij doen om het huiswerk te maken?
- [ ] Welk huiswerk heeft Mary?
- [ ] Op welke dag zijn de lessen?
- [ ] Ik wil het boek Natuurkunde 1 lenen om Nederlands te leren.
- [ ] Ik kan de toets maken en komen op 5 juli om 10.00 uur.
- [ ] Vandaag en morgen kan ik niet naar school komen, omdat ik moet werken en omdat ik ziek ben.
- [ ] Mijn kind Mary kan morgen niet naar school komen, omdat zij ziek is.
- [ ] Ik ga morgen naar school.
- [ ] Ik ging naar school in de stad. Het was een grote school.
- [ ] Ik heb twee lessen per week.
- [ ] Ik maakte mijn huiswerk.
- [ ] Ik studeer Engels in het weekend en ik volg een cursus Nederlands, want ik wil Nederlands leren.
- [ ] Het boek is niet in de bibliotheek en het park is bij mijn huis.
- [ ] U krijgt het boek op 5 juli terug.
- [ ] De studenten lezen boeken. Ze zijn in het park.
- [ ] Dat zijn een tv, een kaart en een klok. Cursisten kunnen leren met de spullen.
- [ ] Hij doet een examen. Hij krijgt een diploma. Hij is blij.
- [ ] Zij is een student. Zij heeft een diploma. Zij is blij.
