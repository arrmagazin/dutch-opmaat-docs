# The Computer

Words and phrases for talking about your computer, the internet, and everyday tech.

## Woordenschat — Vocabulary

- [ ] Mijn **computer** is heel snel.
- [ ] Ik neem mijn **laptop** mee naar werk.
- [ ] Het **scherm** is te donker.
- [ ] Dit **toetsenbord** is nieuw.
- [ ] Waar is de **muis**?
- [ ] Ik zoek het antwoord op **internet**.
- [ ] Ik schrijf een **e-mail** aan mijn baas.
- [ ] Ik ben mijn **wachtwoord** vergeten.
- [ ] Kun je het **bestand** naar mij sturen?
- [ ] De foto's staan in deze **map**.
- [ ] De **printer** is kapot.
- [ ] Deze **website** is heel handig.
- [ ] Ik open een nieuw **programma**.
- [ ] Druk op de rode **knop**.
- [ ] Ik moet dit document nog **opslaan**.
- [ ] Ik wil deze app **downloaden**.
- [ ] Je moet twee keer **klikken**.
- [ ] Je kunt hier **inloggen**.

## Uitdrukkingen — Common phrases

- [ ] 's Ochtends moet ik eerst **de computer opstarten**.
- [ ] Je kunt **inloggen met je wachtwoord**.
- [ ] Vergeet niet om je **bestand op te slaan**.
- [ ] Je moet **op de knop klikken** om te beginnen.
- [ ] Ik ga nu een **e-mail versturen**.
- [ ] Ik kan geen **verbinding maken met internet**.
- [ ] Mijn **computer loopt vast**.
- [ ] Help, **ik ben mijn wachtwoord vergeten**!
- [ ] Ik wil een film **downloaden** van internet.
- [ ] Sorry, **de printer doet het niet**.
- [ ] Ik gebruik een computer en het internet voor mijn werk om e-mails te schrijven, en ik gebruik mijn telefoon `enum:often:elke dag` om te bellen.
- [ ] Zij werkt op haar computer. Zij heeft een glas water. Zij heeft een reactie.
