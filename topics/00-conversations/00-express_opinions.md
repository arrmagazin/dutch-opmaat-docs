# Connecting to speaker

## starters

- [ ] **Ik vind dat** we moeten gaan.
- [ ] **Ik denk dat** het morgen regent.
- [ ] **Volgens mij** heb je gelijk.
- [ ] **Naar mijn idee** is het waar.
- [ ] **In mijn ogen** heb je ongelijk.
- [ ] **Persoonlijk** vind ik het lekker.
- [ ] **Wat mij betreft** hoeft dat niet.

## Affirmatives and Certainty

- [ ] Hij komt **zeker**.
- [ ] **Natuurlijk**, ik help je.
- [ ] **Precies**! Zo bedoel ik.
- [ ] Dat is **eigenlijk** / **grotendeels** waar.
- [ ] ja hoor / ja graag
- [ ] Heel goed gedaan
- [ ] Ja, **dat klopt** helemaal.
- [ ] Dat dacht ik al.
- [ ] Komt het goed? -- **Ik denk(hoop) van wel**.
- [ ] Regent het morgen? -- **Ik denk van niet**.
- [ ] Dat geeft niet.
- [ ] Tja, **zo is het** nu eenmaal.
- [ ] **Dat spreekt** vanzelf.
- [ ] **Uiteraard** ga ik mee.
- [ ] **Inderdaad**, dat is waar.
- [ ] Daar ben ik het **mee eens**.

## Possibility and Doubts

- [ ] **Misschien** regent het straks.
- [ ] Hij is **waarschijnlijk** thuis.
- [ ] **Het is mogelijk** dat ik later kom.
- [ ] Ja hoor, **dat kan**.
- [ ] Dat **lijkt me** goed.
- [ ] Komt hij ook? -- **Dat zou kunnen**.
- [ ] **Wie weet** zien we elkaar nog eens.
- [ ] **Ik weet het niet zeker**, vraag het hem maar.

## Negation and Hedging Phrases

- [ ] Ik vind het **niet echt** lekker.
- [ ] **Nee hoor**. **Echt niet**, nooit.
- [ ] Ik wil **liever niet** gaan.
- [ ] Het is **niet zo** moeilijk.
- [ ] Hij is **niet bepaald** vriendelijk.
- [ ] **Integendeel**, het ging goed.
- [ ] Dat is **eerder** een voordeel.

## Attitude

- [ ] Ik zou willen dat er minder lawaai is van het verkeer in de avond.
- [ ] Verder ben ik heel tevreden.
- [ ] **Ik hou van** koffie.
- [ ] **Ik ben dol op** chocolade.
- [ ] Het **belangrijkste** voor mij is dat...
- [ ] **Ik heb een hekel aan** wachten.
- [ ] **Ik kan** dat geluid **niet uitstaan**.
- [ ] **Ik geef de voorkeur aan** thee.
- [ ] **Ik ben gek op** reizen.
- [ ] **Het kan me niet schelen** wat hij zegt.
- [ ] **Ik ben benieuwd** naar de uitslag.
- [ ] **Ik heb er (geen) zin in**!

### graag → liever → liefst

- [ ] Ik lees **graag** in de tuin.
— [ ] **Ik heb liever** koffie.
— [ ] Ik kom **liefst** morgen.
- [ ] Ik **zou graag** extra les willen.

- [ ] Ik woon (graag|liever) in de stad, want (het is `enum:goed:gezellig` en er is veel te doen).
- [ ] Ik werk graag in de bibliotheek met mijn tien collega's, want het is `enum:goed:rustig`.
- [ ] Ik lees graag boeken en romans in het park, want het is `enum:goed:rustig`.
- [ ] Ik maak (mijn huiswerk graag in een café, want het is `enum:goed:gezellig` | het liefst pasta: ik kook de pasta, snijd tomaten en meng alles samen).
- [ ] Ik wil (graag `enum:drink`, want het is `enum:goed:leuk` | graag een boek lenen en samen iets doen | graag leren koken op internet en yoga leren bij een sportschool | graag naar Japan gaan, want het is `enum:goed:interessant` | liever barman zijn, want het is `enum:goed:leuk`).

- [ ] Ik ben graag thuis, maar in het weekend slaap ik graag lang.
- [ ] Ik hoor graag van u.

- [ ] Ik doe dat liever in een kleine groep, want het is `enum:goed:leuk`.
- [ ] Ik heb liever les in het oude gebouw, want het is `enum:goed:gezellig`.

- [ ] Ik draag het liefst jurken de hele dag.

### ik vind

- [ ] Ik vind voetbal **leuk**.
- [ ] **Ik vind het prima** zo.
- [ ] Ik vind dat de groep te groot is en ik leer niet genoeg.
- [ ] Ik vind documentaires, Nederlandse muziek, voetbal en yoga leuk, maar ik vind realityshows, jazzmuziek en zwemmen niet leuk.
- [ ] Ik vind Nederlandse en Italiaanse restaurants allebei leuk.
- [ ] Ik vind fietsen in Nederland leuk, want het is `enum:goed:gezond`, en Koningsdag vind ik `enum:goed:gezellig` en `enum:goed:interessant`.
- [ ] Ik vind (zwemmen leuk om te kijken | groentesoep lekker), want het is (`enum:goed:spannend` | `enum:goed:gezond`).
- [ ] Ik vind (een café een fijne plek om te leren | les in het klaslokaal fijner), want het is (`enum:goed:gezellig` | `enum:goed:rustig`).
- [ ] Ik vind de zomer een fijn seizoen en vrijdag een fijne dag om boodschappen te doen, want dan is het `enum:goed:warm` en `enum:goed:rustig`.
- [ ] Ik vind mijn buren fijn en ik ken ze nu drie jaar.
- [ ] Ik vind online leren fijn, maar ik mis de docent.
- [ ] Ik vind het leuk dat ik ben uitgenodigd.

> After Ik denk dat / Ik vind dat the verb moves to the end of the clause: *Ik denk dat het leuk is* — not *…dat het is leuk*.
>
