# Clothing

Words and phrases for talking about clothes and getting dressed.

## Woordenschat — Vocabulary

> Vocabulary:

- [ ] Ik koop nieuwe **kleding** voor de winter.
- [ ] Deze **broek** is te lang voor mij.
- [ ] Hij draagt een wit **overhemd** naar zijn werk.
- [ ] Het is koud, doe je **trui** aan.
- [ ] Vergeet je **jas** niet, het regent.
- [ ] Zij draagt een mooie blauwe **jurk**.
- [ ] Deze **rok** past goed bij die blouse.
- [ ] In de zomer draag ik vaak een **T-shirt**.
- [ ] Mijn **schoenen** zijn nat van de regen.
- [ ] Ik zoek nog een paar schone **sokken**.
- [ ] Doe een **sjaal** om, het is koud buiten.
- [ ] Zonder **muts** krijg ik het koud aan mijn hoofd.
- [ ] Ik draag **handschoenen** als het vriest.
- [ ] Deze broek heeft een **riem** nodig.
- [ ] Ik doe schoon **ondergoed** aan.
- [ ] Welke **maat** heeft u nodig, medium of large?
- [ ] Ik ga mijn schoenen **aantrekken**.
- [ ] Wil je je jas **uittrekken**?

## Uitdrukkingen — Common phrases

- [ ] **Wat trek je aan** naar het feest?
- [ ] **Mag ik het passen** in de paskamer?
- [ ] Deze jas **past goed**. Die jurk **staat je mooi**.
- [ ] Deze broek **is te klein**. De trui **is te groot** voor mij.
- [ ] Heeft u dit in een **andere kleur**?
- [ ] Ik **draag** deze kleren op maandag.
- [ ] Mijn **jurken** zijn zwart.
