# Groceries

Words and phrases for shopping for food at the supermarket and the market.

## Woordenschat — Vocabulary

- [ ] Ik heb niet veel **geld** bij me.
- [ ] Dat kost tien **euro**.
- [ ] Kan ik met de kaart **betalen**?
- [ ] Kan ik hier **pinnen**?
- [ ] Ik betaal liever niet **contant**.
- [ ] De **prijs** is te hoog.
- [ ] De **kosten** zijn dit jaar hoger.
- [ ] Deze schoenen zijn te **duur**.
- [ ] De trein is nu **goedkoop**.
- [ ] Op deze kaas krijg je tien procent **korting**.
- [ ] Mag ik de **rekening**, alstublieft?
- [ ] Wilt u de **bon** meenemen?
- [ ] De koffie is in de **aanbieding** deze week.
- [ ] Ik ga vanmiddag **boodschappen** doen.
- [ ] Ik heb alles op mijn **boodschappenlijstje** gezet.

- [ ] Ik ga naar de **bank** voor een nieuwe pas.
- [ ] Het geld staat op mijn **spaarrekening**.
- [ ] Ik betaal elke maand de **huur**.
- [ ] We hebben maar een klein **budget**.
- [ ] Mijn **uitgaven** zijn deze maand hoog.
- [ ] Zijn **inkomen** is niet zo hoog.
- [ ] We betalen de **lening** in vijf jaar terug.
- [ ] We **sparen** voor een vakantie.
- [ ] Mag ik wat geld van je **lenen**?

- [ ] De **supermarkt** is open tot acht uur.
- [ ] Ik pak een **winkelmandje** bij de ingang.
- [ ] Voor veel spullen neem ik een **winkelwagen**.
- [ ] We staan in de rij bij de **kassa**.
- [ ] Bij de **bakker** koop ik vers brood.
- [ ] De **slager** heeft lekkere worst.
- [ ] De **groenteboer** verkoopt verse groente.
- [ ] Tandpasta koop ik bij **de drogist**.
- [ ] Op zaterdag gaan we naar de **markt**.
- [ ] De pasta ligt in dat **schap**.
- [ ] Op de **verpakking** staat hoeveel het weegt.
- [ ] Kijk even naar de **houdbaarheidsdatum**.

## Uitdrukkingen — Common phrases

- [ ] **Hoeveel kost het** in totaal?
- [ ] Nee, **dat is te duur** voor mij.
- [ ] Kijk, **dat is goedkoop**!
- [ ] **Kan ik pinnen** bij de kassa?
- [ ] Betaalt u **contant of met de kaart**?
- [ ] Ik wil minder **geld uitgeven**.
- [ ] We proberen elke maand **geld te sparen**.
- [ ] Van dit inkomen kunnen we goed **rondkomen**.
- [ ] Zullen we **de rekening delen**?
- [ ] Deze maand moeten we **op de kleintjes letten**.

- [ ] **Wat kost dit** per kilo?
- [ ] **Waar staat de** melk?
- [ ] **Waar vind ik** de melk?
- [ ] **Heeft u dat ook** in een grotere maat?
- [ ] **Mag ik afrekenen** bij deze kassa?
- [ ] **Doet u er maar een pond**, alstublieft.
- [ ] Betaalt u **contant of pinnen**?
- [ ] **Wilt u een tasje** erbij?
- [ ] **Mag ik een kilo** appels?
- [ ] Wat is de oplossing en wat kunt u doen?
- [ ] Wat zijn de openingstijden?
- [ ] Of kunt u het pakket bij mijn deur leggen?
- [ ] Kan ik een nieuw product krijgen?
- [ ] Ik doe twee keer per week mijn boodschappen in de supermarkt en ik koop tomaten en paprika.
- [ ] Ik heb een boek gekocht in een boekwinkel, maar ik heb ook een boek geleend dat beschadigd is en een product ontvangen dat kapot is.
- [ ] Ik krijg het terug op [datum].
- [ ] Ik zag het toen ik het pakket opende en toen ik het las.
- [ ] Ja, ik koop vaak online, want het is `enum:goed:makkelijk`.
- [ ] Dat zijn tassen. Je kan er spullen in doen.
- [ ] De vrouw winkelt. Zij is in de supermarkt. Zij heeft melk.
