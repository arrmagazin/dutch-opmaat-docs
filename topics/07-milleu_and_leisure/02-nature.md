# Nature

Words and phrases for talking about nature, the outdoors, and the landscape.

## The Weather

> Vocabulary:
  `table:/topic_weather?columns=name_en#Weather`

Words and phrases for talking about the weather in Dutch.

- [ ] Hoe is het **weer** vandaag?
- [ ] De **zon** schijnt fel vandaag.
- [ ] Ik hou niet van de **regen**.
- [ ] Er is geen enkele **wolk** aan de hemel.
- [ ] De **wind** is vandaag heel koud.
- [ ] De kinderen spelen in de **sneeuw**.
- [ ] Vannacht was er zwaar **onweer**.
- [ ] Er komt een **bui** aan, neem een jas mee.
- [ ] De **temperatuur** daalt vanavond.
- [ ] Het is vandaag twintig **graden**.
- [ ] Volgens het **weerbericht** wordt het mooi.
- [ ] Vergeet je **paraplu** niet.
- [ ] Het is lekker **warm** buiten.
- [ ] In de winter is het vaak **koud**.
- [ ] Het gaat straks **regenen**.
- [ ] Het kan aan zee hard **waaien**.
- [ ] Het begint zachtjes te **sneeuwen**.
- [ ] Vannacht gaat het **vriezen**.

Uitdrukkingen — Common phrases

- [ ] **Wat een mooi weer** vandaag!
- [ ] Vandaag **is het lekker zonnig**.
- [ ] Vanmorgen **is het bewolkt**.
- [ ] Neem een paraplu mee, **het regent hard**.
- [ ] Trek een jas aan, **het is koud buiten**.
- [ ] **Wat is het warm** vandaag!
- [ ] Pas op, **het waait hard**.
- [ ] **Hoeveel graden is het** buiten?
- [ ] Kijk, **het klaart op**.
- [ ] Het wordt koud, **neem een jas mee**.
- [ ] Het regent, dus ik neem mijn jas mee.

## Plants

> Vocabulary:
  `table:/topic_vegetables?columns=name_en#vegetables`
  `table:/topic_tree?columns=name_en#tree`

- [ ] In het weekend zijn we graag in de **natuur**.
- [ ] Onder de **boom** is het lekker koel.
- [ ] Ik heb een mooie **bloem** geplukt.
- [ ] De kinderen spelen in het **gras**.
- [ ] We maken een wandeling door het **bos**.
- [ ] Deze **plant** heeft veel water nodig.
- [ ] In de herfst valt elk **blad** van de boom.
- [ ] De **rivier** stroomt door het dorp.
- [ ] In de zomer zwemmen we in het **meer**.
- [ ] Vanaf de **berg** heb je een mooi uitzicht.
- [ ] We liggen de hele dag op het **strand**.
- [ ] De **zee** is vandaag heel rustig.
- [ ] De **lucht** is vandaag helderblauw.
- [ ] De **zon** schijnt de hele dag.
- [ ] Vannacht is de **maan** heel helder.
- [ ] Aan de hemel zie ik een heldere **ster**.
- [ ] Welk **dier** is dat daar in het bos?
- [ ] We volgen het **pad** langs de rivier.

Uitdrukkingen — Common phrases

- [ ] Zullen we een **wandeling maken** in het park?
- [ ] We willen even **van de natuur genieten**.
- [ ] Het weer is mooi, laten we **naar buiten gaan**.
- [ ] **Wat een mooi uitzicht** vanaf hier!
- [ ] Ik loop graag **in de frisse lucht**.
- [ ] **De zon schijnt** en het is warm.
- [ ] In de ochtend **fluiten de vogels**.
- [ ] Op zondag **gaan we naar het bos**.
- [ ] Ze wonen graag **dicht bij de natuur**.
- [ ] Ik ga elke dag naar het park en ik wandel er in mijn vrije tijd, want het is `enum:goed:rustig`.

## Animals

> Vocabulary:
  `table:/topic_species?columns=name_en#species`
  `table:/topic_tamed?columns=name_en#tamed`
  `table:/topic_animals?columns=name_en#animals`
  `table:/topic_bird?columns=name_en#Birds`

Words and phrases for talking about animals, pets and the zoo.

- [ ] Een kat is een populair **huisdier**.
- [ ] Een hond is een lief **dier**.
- [ ] Onze **hond** speelt in de tuin.
- [ ] De **kat** slaapt op de bank.
- [ ] Het **konijn** eet een wortel.
- [ ] Een **vogel** zingt in de boom.
- [ ] We hebben een **vis** in een kom.
- [ ] Het **paard** rent over het veld.
- [ ] De **koe** staat in de wei.
- [ ] Het **schaap** heeft witte wol.
- [ ] Het **varken** ligt in de modder.
- [ ] De **kip** legt elke dag een ei.
- [ ] De **eend** zwemt in de vijver.
- [ ] Een kleine **muis** loopt over de vloer.
- [ ] De **olifant** is heel groot.
- [ ] De **leeuw** slaapt de hele dag.
- [ ] De **aap** klimt in de boom.
- [ ] In de **dierentuin** zien we veel **dieren**.

Uitdrukkingen — Common phrases

- [ ] **Heb je huisdieren** thuis? -- **Ik heb een hond** en een kat.
- [ ] Ik ga **de hond uitlaten** in het park.
- [ ] Vergeet niet de kat **eten te geven**.
- [ ] **Wat voor dier is dat** in de boom?
- [ ] Mijn zus is **bang voor** grote honden.
- [ ] We willen morgen **naar de dierentuin gaan**.
- [ ] Kinderen leren **een dier te verzorgen**.
