# Dutch Culture

Words and phrases for talking about Dutch culture, traditions and typical customs.

## Woordenschat — Vocabulary

- [ ] Op **Koningsdag** draagt iedereen oranje.
- [ ] Kinderen krijgen cadeaus van **Sinterklaas**.
- [ ] Bij het dorp staat een oude **molen**.
- [ ] Vroeger droegen boeren een **klomp** van hout.
- [ ] In de lente bloeit de **tulp** in het veld.
- [ ] Ik ga met de **fiets** naar mijn werk.
- [ ] Nederlandse **kaas** is heel bekend.
- [ ] Veel mensen eten graag een **haring**.
- [ ] Langs de **gracht** staan mooie huizen.
- [ ] In het museum hangt een oud **schilderij**.
- [ ] We gaan naar het **museum** in Amsterdam.
- [ ] Sinterklaas is een oude **traditie**.
- [ ] Op Koningsdag is er overal een **feest**.
- [ ] Drie zoenen geven is een Nederlandse **gewoonte**.
- [ ] Iedereen draagt **oranje** kleren op Koningsdag.
- [ ] Het is heel **gezellig** bij ons thuis.
- [ ] Wil je een **dropje**? Het is echt Nederlands.
- [ ] Bij de koffie eet ik een **stroopwafel**.

## Uitdrukkingen — Common phrases

- [ ] **Fijne Koningsdag** allemaal!
- [ ] **Wat gezellig** dat je er bent!
- [ ] Op de fiets naar je werk is **typisch Nederlands**.
- [ ] Ik ga altijd **op de fiets** naar school.
- [ ] We gaan **Sinterklaas vieren** met de familie.
- [ ] Ze wil een **feestje geven** voor haar verjaardag.
- [ ] In Nederland is het gewoon om **drie zoenen te geven**.
- [ ] **Doe maar normaal**, zeggen Nederlanders vaak.
- [ ] Deze stroopwafel is **lekker hoor**!
- [ ] Schaatsen op het ijs is **een echte traditie**.

## Feesten & uitnodigen — Celebrations & inviting

- [ ] We zijn uitgenodigd voor **een bruiloft**.
- [ ] **Kom je naar mijn feest** op zaterdag?
- [ ] Volgende week **geef ik een feestje**.
- [ ] **Bedankt voor de uitnodiging**, ik kom graag.
- [ ] **Gefeliciteerd** met je nieuwe baan!
- [ ] Ik heb **een cadeau** voor haar gekocht.
- [ ] Ik ga haar **een kaartje sturen**.
- [ ] Zullen we **bloemen meenemen** voor de gastvrouw?
- [ ] Ik vier mijn verjaardag elk jaar en ik vier het feest met mijn man.
- [ ] Mijn vriend komt op het feest.

## Holiday

Words and phrases for talking about your holiday, travel, and time away.

- [ ] We gaan in de zomer op **vakantie**.
- [ ] De **reis** naar Spanje duurt drie uur.
- [ ] Ik **reis** graag met de trein.
- [ ] We nemen het **vliegtuig** naar Italië.
- [ ] Het **vliegveld** is ver van de stad.
- [ ] Ons **hotel** ligt vlak bij het strand.
- [ ] We staan met de tent op een **camping**.
- [ ] 's Nachts slapen we in de **tent**.
- [ ] De kinderen spelen op het **strand**.
- [ ] De **zee** is vandaag heel rustig.
- [ ] In de zomer schijnt de **zon** veel.
- [ ] Mijn **koffer** is te zwaar.
- [ ] Ik moet mijn koffer nog **inpakken**.
- [ ] Vergeet je **paspoort** niet!
- [ ] Ik heb het **ticket** op mijn telefoon.
- [ ] We **boeken** het hotel online.
- [ ] Wat is jullie **bestemming**?
- [ ] Ik stuur oma een **ansichtkaart**.

Uitdrukkingen — Common phrases

- [ ] **Fijne vakantie**, tot volgende week!
- [ ] **Goede reis**, bel me als je er bent.
- [ ] **Waar ga je naartoe** deze zomer?
- [ ] Volgende maand **ga ik op vakantie**.
- [ ] **Hoe lang blijf je** in Frankrijk?
- [ ] We willen **een kamer boeken** voor twee nachten.
- [ ] **Hoeveel kost het** ticket naar Berlijn?
- [ ] Een mooie reis, **geniet ervan**!
- [ ] De vakantie begint bijna, **ik heb er zin in**!
- [ ] **Tot ziens**, tot na de vakantie!
