# The Computer

Words and phrases for talking about your computer, the internet, and everyday tech.

## Woordenschat — Vocabulary

| Nederlands | English | Example | ExampleEn |
|------------|---------|---------|---------|
| De computer | The computer | Mijn **computer** is heel snel. | My computer is very fast. |
| De laptop | The laptop | Ik neem mijn **laptop** mee naar werk. | I take my laptop to work. |
| Het scherm | The screen | Het **scherm** is te donker. | The screen is too dark. |
| Het toetsenbord | The keyboard | Dit **toetsenbord** is nieuw. | This keyboard is new. |
| De muis | The mouse | Waar is de **muis**? | Where is the mouse? |
| Het internet | The internet | Ik zoek het antwoord op **internet**. | I look up the answer on the internet. |
| De e-mail | The email | Ik schrijf een **e-mail** aan mijn baas. | I'm writing an email to my boss. |
| Het wachtwoord | The password | Ik ben mijn **wachtwoord** vergeten. | I forgot my password. |
| Het bestand | The file | Kun je het **bestand** naar mij sturen? | Can you send me the file? |
| De map | The folder | De foto's staan in deze **map**. | The photos are in this folder. |
| De printer | The printer | De **printer** is kapot. | The printer is broken. |
| De website | The website | Deze **website** is heel handig. | This website is very useful. |
| Het programma | The program | Ik open een nieuw **programma**. | I open a new program. |
| De knop | The button | Druk op de rode **knop**. | Press the red button. |
| Opslaan | To save | Ik moet dit document nog **opslaan**. | I still need to save this document. |
| Downloaden | To download | Ik wil deze app **downloaden**. | I want to download this app. |
| Klikken | To click | Je moet twee keer **klikken**. | You have to click twice. |
| Inloggen | To log in | Je kunt hier **inloggen**. | You can log in here. |

## Uitdrukkingen — Common phrases

| Nederlands | English | Example | ExampleEn |
|------------|---------|---------|---------|
| De computer opstarten | To start up the computer | 's Ochtends moet ik eerst **de computer opstarten**. | In the morning I first have to start up the computer. |
| Inloggen met je wachtwoord | To log in with your password | Je kunt **inloggen met je wachtwoord**. | You can log in with your password. |
| Een bestand opslaan | To save a file | Vergeet niet om je **bestand op te slaan**. | Don't forget to save your file. |
| Op de knop klikken | To click the button | Je moet **op de knop klikken** om te beginnen. | You have to click the button to start. |
| Een e-mail versturen | To send an email | Ik ga nu een **e-mail versturen**. | I'm going to send an email now. |
| Verbinding maken met internet | To connect to the internet | Ik kan geen **verbinding maken met internet**. | I can't connect to the internet. |
| De computer loopt vast | The computer freezes | Mijn **computer loopt vast**. | My computer keeps freezing. |
| Ik ben mijn wachtwoord vergeten | I forgot my password | Help, **ik ben mijn wachtwoord vergeten**! | Help, I forgot my password! |
| Iets downloaden | To download something | Ik wil een film **downloaden** van internet. | I want to download a movie from the internet. |
| De printer doet het niet | The printer isn't working | Sorry, **de printer doet het niet**. | Sorry, the printer isn't working. |
